## [0.2.0] — Sprint 1 — Calculator

### Added
- Calculator utility under apps/Calculator
- Four arithmetic operations: addition, subtraction, multiplication, division
- Input validation with re-prompt on invalid input
- Division by zero protection via exception handling
- Menu-driven console interface
- Multi-file project structure: calculator.hpp, calculator.cpp, main.cpp
- Calculator-specific CMakeLists.txt integrated into root build
- Calculator README.md with usage and structure documentation
